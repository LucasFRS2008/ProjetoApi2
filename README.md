Scanner de Câmera & OCR em Tempo Real
Descrição
Aplicação web desenvolvida para captura de imagem e transcrição de texto automática. O projeto utiliza o acesso direto ao hardware de vídeo do dispositivo para ler documentos e converter caracteres em texto digital editável.

Funcionalidades
Controle de Câmera: Alternância entre câmera frontal e traseira via facingMode.

Captura e Transcrição: Processamento de imagem para texto em tempo real.

Interface Responsiva: Design otimizado para dispositivos móveis e desktops.

Tecnologias Utilizadas
HTML: Elementos de vídeo e canvas.

CSS: Estilização e layout adaptável.

JavaScript: Manipulação de streams de mídia e lógica de aplicação.
